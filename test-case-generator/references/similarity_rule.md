> 功能点相似度分析规则

**相似度计算规则**:

1. **关键词匹配**(权重 40%)
   - 提取功能点名称和描述的关键词
   - 计算 Jaccard 相似度：`sim = |A∩B| / |A∪B|`

2. **语义匹配**(权重 60%)
   - 使用 TF-IDF 向量化功能点描述
   - 计算余弦相似度：`sim = cos(A, B)`

3. **综合相似度**:
   ```
   similarity = 0.4 * keyword_sim + 0.6 * semantic_sim
   ```

**分组阈值**:

- `similarity >= 0.7`: 判定为相同功能点，归为一组
- `0.5 <= similarity < 0.7`: 可疑重复，标记供人工审核
- `similarity < 0.5`: 判定为不同功能点
