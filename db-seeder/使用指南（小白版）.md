# Database Seeder 使用指南（小白版）

## 📚 这个技能是做什么的？

简单来说，这个技能可以帮你**自动生成大量的测试数据**，然后填充到你的数据库中。

想象一下，你在开发一个电商网站，需要测试用户注册、商品展示、订单处理等功能。如果手动一条条创建测试数据，那太累了！这个技能可以帮你：
- ✅ 自动生成100个用户
- ✅ 自动生成1000个商品
- ✅ 自动生成5000个订单
- ✅ 而且数据看起来很真实（比如真实的姓名、地址、电话号码等）

## 🎯 什么时候需要用它？

### 场景1：刚开始开发项目
你需要一些测试数据来验证功能是否正常工作。

### 场景2：写自动化测试
你需要固定的测试数据来确保测试结果一致。

### 场景3：演示给客户看
你需要一些看起来真实的数据来展示系统功能。

### 场景4：测试数据库性能
你需要大量数据来测试数据库查询速度。

## 🚀 快速上手（3步搞定）

### 第1步：安装必要的工具

打开命令行，输入以下命令：

```bash
# 安装核心工具
pip install faker pyyaml

# 根据你的数据库类型安装对应的驱动
# 如果是 PostgreSQL
pip install sqlalchemy psycopg2-binary

# 如果是 MySQL
pip install sqlalchemy pymysql

# 如果是 MongoDB
pip install pymongo
```

### 第2步：生成测试数据

有两种方式：

#### 方式A：使用模板生成（推荐新手）

```bash
# 生成面试系统的测试数据
python .claude/skills/db-seeder/scripts/generate_fixtures.py \
  --template elios-interview \
  --output fixtures/test_data.json \
  --pretty
```

#### 方式B：自定义生成

```bash
# 生成100个用户和500篇文章
python .claude/skills/db-seeder/scripts/generate_fixtures.py \
  --models User:100,Post:500 \
  --output fixtures/test_data.json \
  --pretty
```

### 第3步：填充到数据库

```bash
# PostgreSQL
python .claude/skills/db-seeder/scripts/seed_database.py \
  --fixtures fixtures/test_data.json \
  --db postgresql \
  --connection "postgresql://用户名:密码@localhost:5432/数据库名"

# MySQL
python .claude/skills/db-seeder/scripts/seed_database.py \
  --fixtures fixtures/test_data.json \
  --db mysql \
  --connection "mysql://用户名:密码@localhost:3306/数据库名"

# SQLite
python .claude/skills/db-seeder/scripts/seed_database.py \
  --fixtures fixtures/test_data.json \
  --db sqlite \
  --connection "sqlite:///./test.db"
```

## 📖 详细使用说明

### 1. 自动检测数据库配置

如果你不知道数据库连接信息，可以让工具自动检测：

```bash
python .claude/skills/db-seeder/scripts/detect_db_config.py
```

这个命令会：
- 扫描你的项目配置文件
- 查找数据库连接信息
- 显示可用的数据库连接

### 2. 检查数据库结构（高级功能）

如果你想根据现有的数据库表结构生成数据：

```bash
# 查看数据库中有哪些表
python .claude/skills/db-seeder/scripts/inspect_schema.py \
  --db postgresql \
  --connection "postgresql://user:pass@localhost/mydb"

# 自动生成工厂函数和测试数据模板
python .claude/skills/db-seeder/scripts/inspect_schema.py \
  --db postgresql \
  --connection "postgresql://user:pass@localhost/mydb" \
  --generate-factories \
  --generate-fixtures
```

### 3. 使用配置文件（适合复杂场景）

如果你想更灵活地控制数据生成：

```bash
# 1. 复制配置模板
cp assets/seed-config-template.yaml my-seed-config.yaml

# 2. 编辑配置文件（用记事本或VS Code打开）
# 修改数据库连接、模型定义、数据数量等

# 3. 使用配置文件生成数据
python .claude/skills/db-seeder/scripts/seed_database.py \
  --config my-seed-config.yaml
```

配置文件示例（my-seed-config.yaml）：

```yaml
database:
  type: mysql
  connection: mysql://root:password@localhost:3306/mydb

faker:
  locale: zh_CN  # 使用中文生成数据

models:
  - name: User
    count: 100
    factory:
      username: "lambda fake: fake.user_name()"
      email: "lambda fake: fake.email()"
      name: "lambda fake: fake.name()"
```

## 💡 实用技巧

### 技巧1：生成可重复的数据

如果你每次运行都想生成相同的数据（方便测试）：

```python
from faker import Faker

# 设置种子
Faker.seed(12345)
fake = Faker()

# 每次运行都会生成相同的数据
print(fake.name())  # 总是输出相同的名字
```

### 技巧2：批量插入大量数据

如果数据量很大，建议分批插入：

```yaml
# 在配置文件中设置
options:
  batch_size: 1000  # 每1000条提交一次
```

### 技巧3：避免重复数据

```python
# 使用 unique=True 确保数据不重复
skills: "lambda fake: fake.random_elements(['Python', 'Java', 'Go'], length=3, unique=True)"
```

### 技巧4：生成关联数据

如果你的表之间有外键关系，先生成父表数据，再生成子表数据：

```yaml
models:
  - name: User  # 先生成用户
    count: 100
    
  - name: Post  # 再生成文章，引用用户ID
    count: 500
    factory:
      author_id: "lambda fake: fake.random_int(min=1, max=100)"
```

## 🔧 常见问题解决

### 问题1：提示"Faker未安装"

**解决方法：**
```bash
pip install faker
```

### 问题2：数据库连接失败

**解决方法：**
1. 检查数据库是否启动
2. 检查用户名、密码是否正确
3. 检查数据库名称是否存在
4. 检查端口是否正确（PostgreSQL默认5432，MySQL默认3306）

### 问题3：提示"唯一约束冲突"

**解决方法：**
1. 清空数据库中的旧数据
2. 或者使用 `skip_if_exists: true` 选项跳过已存在的数据

### 问题4：生成速度太慢

**解决方法：**
1. 减少数据量
2. 增加批次大小（batch_size）
3. 使用更快的数据库（如SQLite）


## 🎓 进阶使用

### 自定义数据生成规则

如果你需要生成特定的数据格式，可以自定义工厂函数：

```python
# 创建自定义脚本 custom_seeder.py
from faker import Faker

fake = Faker()

def generate_custom_user():
    return {
        'username': fake.user_name(),
        'email': fake.email(),
        # 自定义规则：手机号必须以1开头
        'phone': '1' + fake.msisdn()[3:],
        # 自定义规则：年龄在18-60之间
        'age': fake.random_int(min=18, max=60),
    }

# 使用自定义函数生成数据
users = [generate_custom_user() for _ in range(100)]
```
